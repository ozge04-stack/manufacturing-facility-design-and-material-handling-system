# Manufacturing Facility Design and Material Handling System

This project focuses on the design and analysis of a manufacturing facility for the production of rifle magazine and adjustable cheek rest components. The study was conducted as part of the **Facility Design and Material Handling Systems** course.

The project integrates several industrial engineering methodologies including make-or-buy analysis, assembly line balancing, cellular manufacturing, facility layout planning, and simulation-based performance analysis.

## Project Objectives

The main goal of the project is to design an efficient production system that minimizes operational costs while maintaining high productivity and smooth material flow.

Key objectives include:

- Determining make-or-buy decisions for product components
- Designing product structures and process flow
- Calculating machine and operator requirements
- Balancing assembly lines using different line configurations
- Designing cellular manufacturing systems
- Developing facility layout alternatives
- Optimizing material handling systems
- Evaluating system performance through simulation

## Products Analyzed

The facility is designed for the production of two main components used in the **Hatsan Escort ERX rifle system**:

- Rifle Magazine
- Adjustable Cheek Rest

These products include several subcomponents such as:

- Magazine body
- Base plate
- Cartridge guide
- Cheek support unit
- Piston and ring components

## Methods and Techniques Used

The project applies multiple industrial engineering tools and methods:

### Production Planning

- Make-or-Buy Analysis
- Product Tree (Bill of Materials)
- Process Flow Analysis

### Capacity Planning

- Machine Requirement Calculation
- Operator Requirement Analysis
- Bottleneck Identification

### Assembly Line Balancing

Two line configurations were analyzed:

- Straight Line Balancing
- U-Shaped Line Balancing

Performance indicators evaluated:

- Line Efficiency
- Balance Delay
- Smoothness Index
- Idle Time

Optimization models were also validated using **LINGO**.

### Cellular Manufacturing

To improve production efficiency:

- Binary Ordering Algorithm
- Single Pass Heuristic

These methods were used to create production cells and minimize inter-cell material flow.

### Facility Layout Planning

Facility layout was designed using:

- From-To Flow Matrix
- Activity Relationship Chart (SLP – Systematic Layout Planning)
- Alternative block layouts

Different layout alternatives were evaluated based on material flow cost.

### Material Handling System Design

The project includes:

- Transport equipment selection
- Unit load analysis
- Trip frequency calculation
- Equipment requirement estimation
- Material handling cost analysis

### Simulation Analysis

A simulation model was developed to evaluate system performance under different operational scenarios.

The simulation analyzes:

- Equipment utilization
- Waiting times
- Bottleneck behavior
- Transportation capacity
- Operational costs

Three scenarios were tested:

- Base Scenario
- Intermediate Scenario (+30% demand)
- Stress Scenario (+50% demand)

## Key Findings

Results show that:

- U-shaped assembly lines significantly improve efficiency compared to straight-line layouts.
- Cellular manufacturing reduces unnecessary material movement.
- Proper equipment allocation eliminates bottlenecks.
- Simulation-based analysis helps evaluate operational risks under demand fluctuations.
